CI CD Deployment on Azure AKS

This application is deployed on Azure Kubernetes Service using a GitHub Actions CI CD pipeline. The deployment is fully automated from code commit to production.

Project Overview

I built a simple HTML application, containerized it using Docker, pushed the image to Azure Container Registry, and deployed it to AKS using Kubernetes manifests. Every code change triggers an automated build and deployment through GitHub Actions.

Technology Stack

Azure Kubernetes Service Azure Container Registry GitHub Actions Docker Kubernetes CI CD

Key Learnings

  • End-to-end CI CD pipeline using GitHub Actions
  • Secure image push and pull between ACR and AKS
  • Kubernetes Deployments and Services
  • Basic Azure networking and LoadBalancer concepts
Deployed as part of a hands-on learning project by a Junior Azure Engineer
